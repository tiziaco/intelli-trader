## Sampling Rate

- **After every task commit:** `PYTHONPATH="$PWD" poetry run pytest tests/unit/<touched-domain> -q` plus `poetry run mypy` (~10s combined)
- **After every plan wave:** `PYTHONPATH="$PWD" poetry run pytest tests -q` **and** `PYTHONPATH="$PWD" poetry run mypy`.
  For waves 2–5 additionally `poetry run pytest tests/integration/test_backtest_oracle.py tests/integration/test_okx_inertness.py -q`
- **Before `/gsd-verify-work`:** full suite green (≥2823 passed, 6 skipped), mypy clean, oracle byte-exact at `134 / 46189.87730727451`, inertness green
- **Max feedback latency:** 46 seconds (full suite)

**Every wave that touches the oracle path must run the oracle test in the same commit.** For the D-05/D-19
and D-18 waves the oracle is the *primary* acceptance criterion, not a smoke check.
