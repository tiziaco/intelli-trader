ratchet-AFTER-evaluate ordering. A `TRAILING_STOP` evaluates EXACTLY like a STOP using its current

Ratchet math (05-CONTEXT Specific Ideas):
  Long sell-stop:  stop = HWM - trail (PRICE) | HWM * (1 - trail) (PERCENT); HWM = max closed highs, seed=fill
  Short buy-stop:  stop = LWM + trail (PRICE) | LWM * (1 + trail) (PERCENT); LWM = min closed lows, seed=fill
  Invariant: longs stop non-decreasing; shorts stop non-increasing.
  quantize(stop, instrument, "price") ONLY on the stop used for trigger/fill (D-TRAIL-8).
