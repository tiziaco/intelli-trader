|----------|-------------|
| test code → DB engine | Test constructs a SqlBackend over a container URL; the URL carries ephemeral container credentials, never a production secret |
| config → engine URL | `SqlSettings.engine_url()` is the single URL builder (URL-escapes password / verbatim escape hatch) |

| T-03-02 | Denial of Service | undisposed engine ResourceWarning | mitigate | fixture disposes the backend in `finally` (WR-03); `filterwarnings=["error"]` would otherwise fail the suite (Pitfall 4) |
| T-03-SC | Tampering | pip installs | accept | no package installs this phase — all deps (SQLAlchemy/Alembic/testcontainers/psycopg2) were gated in Phase 1 (research §Package Legitimacy Audit) |
