<objective>
Give every live data provider a uniform surface (VENUE-05 / D-10): a `LiveDataProvider` runtime_checkable Protocol plus a `BaseLiveDataProvider` supplying no-op defaults for the optional streaming/wiring seams, and make `ReplayDataProvider` inherit it. Purpose: uniform provider wiring; the paper/replay path stops being a special case.

New symbols this phase produces (exclude from drift verification): `LiveDataProvider`, `BaseLiveDataProvider`.
