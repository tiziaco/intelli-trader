```python
def read_all(self) -> list[Mapping[str, Any]]:

    ORDERING CONTRACT: ``venue_name`` ASC then ``account_id`` ASC, so the rehydrate order
```

```python
                self._ctx, spec, self._connectors
            )
        return self._memo[key]
```
