coercion returns.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| exchange-registry write key → both reader keys | The `(venue, account_id)` pair is the venue **session** key; a write/read mismatch routes one account's order through another's authenticated session. |
| connector memo key → venue connection | Two accounts collapsing to one memo key means one authenticated connection serving two accounts. |
