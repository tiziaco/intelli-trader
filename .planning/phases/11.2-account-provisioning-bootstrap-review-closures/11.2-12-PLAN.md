<objective>
Give `PortfolioHandler` the public read surface every consumer was reaching past, convert the one
surviving production reach-in, and move `_layer_persisted_overrides`' degrade guard inside its loops
— in both the portfolio arm and the venue arm.

Purpose: the handler already exposes `get_active_portfolios()`, but every consumer that reached for
the private `_portfolios` dict wanted a different question answered — *all* portfolios, or *is this
one registered*. This is also the public read surface the FastAPI layer needs.

The co-located WR-02 defect is worth stating plainly: the portfolio arm's `try` wraps the **whole**
`for` loop, so one poisoned `config_json` silently skips **every portfolio after it in iteration
order**, emitting one warning that names only the first failure — while the docstring claims
per-scope isolation. The venue arm has the identical defect and is fixed in the same edit.

Output: `all_portfolios()` / `has_portfolio()`, one converted call site, and two per-entry guards.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| persisted `config_json` → live portfolio/venue configuration | Untrusted-at-read stored blobs cross into runtime configuration here. |
| private registry → external consumers | An accessor that hands out a mutable alias is an unguarded write boundary. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-11.2-12-01 | Denial of Service | whole-loop degrade guard | high | mitigate | Moving the guard inside both loops means one poisoned `config_json` degrades exactly one entry instead of silently skipping every entry after it; proven by a two-entry, first-poisoned fixture. |
| T-11.2-12-02 | Repudiation | one warning for N silent failures | high | mitigate | Per-entry guards emit per-entry warnings, so the warning count equals the failure count; the docstring claiming per-scope isolation becomes true rather than aspirational. |
| T-11.2-12-03 | Tampering | mutable registry alias handed out by `all_portfolios()` | medium | mitigate | The accessor returns a copy or immutable view; asserted by an explicit mutation-isolation test. |
| T-11.2-12-04 | Tampering | outer guard kept alongside the inner one | medium | mitigate | Explicitly prohibited — a surviving outer `try` reinstates the defect for any exception the inner guard misses. |
| T-11.2-12-SC | Tampering | npm/pip/cargo installs | high | mitigate | No package is installed; `pyproject.toml` untouched. |
