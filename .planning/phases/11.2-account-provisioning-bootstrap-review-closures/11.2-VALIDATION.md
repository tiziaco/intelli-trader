## Test Infrastructure

| Property | Value |
|----------|-------|
| **Framework** | pytest 8.4.2 (`minversion = "8.0"`, `testpaths = ["tests"]`) |
| **Config file** | `pyproject.toml` `[tool.pytest.ini_options]` — `filterwarnings=["error", ...]`, `--strict-markers`, `--strict-config` |
| **Quick run command** | `poetry run pytest tests/unit -q` (~8s) |
| **Full suite command** | `poetry run pytest tests` — **never `make test`** (exports `ITRADER_DISABLE_LOGS=true`, breaks `caplog` assertions; aborts in worktrees on missing `.env`) |
| **Type gate** | `poetry run mypy` (strict over `itrader`, 282 files) |
| **Estimated runtime** | ~42 seconds full suite; ~8 seconds unit only |
| **Measured baseline** | **2879 passed / 6 skipped / 41.8s**; mypy clean over 282 files |

**Marker discipline:** TYPE axis `unit`/`integration`/`slow`/`e2e` auto-applied by folder (`tests/conftest.py`); PURPOSE axis `smoke`/`live` applied by hand. All markers must be declared or `--strict-markers` fails.
