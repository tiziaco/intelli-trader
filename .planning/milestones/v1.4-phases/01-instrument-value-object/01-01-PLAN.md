<objective>
Land the frozen `Instrument` value object in `core/instrument.py` and rewire
`core/money.py::quantize()` to read its rounding scale off a handed-in `Instrument`,
deleting the hard-coded `_INSTRUMENT_SCALES` table. This satisfies INST-01 (Instrument
is the per-symbol precision source; table deleted) and lands the inert INST-03 margin
fields (`maintenance_margin_rate`, `max_leverage`, `settles_funding`) for downstream phases.

Purpose: `Instrument` is the foundational value object every later v1.4 phase
(margin/leverage/shorts/liquidation) consumes. Removing domain state from the rounding
mechanism (D-05) keeps `money.py` pure/stateless.
Output: `core/instrument.py` (new), rewired `core/money.py`, new `test_instrument.py`,
updated `test_money.py`.

**Byte-exact discipline:** This phase re-baselines NOTHING. The D-02a blast radius is
PROVEN (RESEARCH §1): the module function `quantize()` is imported ONLY by
`tests/unit/core/test_money.py`. Every production `.quantize(` is the stdlib
`Decimal.quantize()` *method* (untouched). The production rounding path changes by zero
bytes, so the SMA_MACD oracle (134 trades / `final_equity 46189.87730727451`) holds by
construction — proven downstream in plan 01-03.
</objective>

From itrader/core/money.py (current state — the rewire target, 4 spaces):
```python
__all__ = ["ONE", "to_money", "quantize"]
ONE = Decimal("1")
_DEFAULT_SCALES: dict[str, Decimal] = {
    "price": Decimal("0.01"),
    "quantity": Decimal("0.00000001"),
    "cash": Decimal("0.01"),
}
_INSTRUMENT_SCALES: dict[str, dict[str, Decimal]] = {       # DELETE THIS WHOLE TABLE
    "BTCUSD": {"price": Decimal("0.00000001"),
               "quantity": Decimal("0.00000001"),
               "cash": Decimal("0.01")},
}
def to_money(x: float | int | str | Decimal) -> Decimal:
    return Decimal(str(x))      # D-04 string path — NEVER Decimal(float)
def quantize(value: Decimal, instrument: str, kind: str) -> Decimal:   # str -> Instrument
    scale = _INSTRUMENT_SCALES.get(instrument, _DEFAULT_SCALES).get(
        kind, _DEFAULT_SCALES[kind])
    return value.quantize(scale, rounding=ROUND_HALF_UP)
```
