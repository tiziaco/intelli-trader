<objective>
Memoize the per-bar timestamp-alignment math (`_aligned`) with a bounded `@functools.lru_cache(maxsize=32)`, eliminating the ~8.7% W1 CPU spent recomputing `astimezone`/`replace`/`total_seconds`/modulo identically across strategies within each TIME event — with the function body byte-unchanged (D-01).

Purpose: `_aligned(ts, tf)` is called once per registered strategy per tick, all sharing the same `event.time` — so the same `(ts, tf)` is recomputed N times intra-tick. Memoizing captures those repeats at O(1) with zero numeric change.
Output: A memoized `_aligned` (decorator + decision-tag comment + `import functools`) and three extension tests in the existing `tests/unit/outils/test_time_parser.py` proving output equivalence, memo activity, and bounded memory.
</objective>
